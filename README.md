# blog-images-TTSM 🚀

A centralized **Static Asset Repository** for the **Thinker TechSutra Marathi (TTSM)** blog.

All technical screenshots, command logs, and infrastructure diagrams are stored here and 
embedded directly into blog posts via GitHub raw URLs — keeping WordPress.com free storage 
usage minimal.

---

## 📝 Blog

🌐 [thinkertechsutramarathi.wordpress.com](https://thinkertechsutramarathi.wordpress.com)

**Thinker TechSutra Marathi** publishes DevOps, Linux, and Cloud learning content in 
**two languages** — each as a separate blog category:

- 🇮🇳 **मराठी** — DevOps & Linux concepts explained simply in Marathi for Marathi-speaking students
- 🇬🇧 **English** — The same learning journey documented in English for a wider audience

---

## 🎯 What This Repository Demonstrates

1. **GitOps for Content Management** — treating blog media as version-controlled assets
2. **Storage Optimization** — offloading screenshots and logs away from WordPress free-tier storage
3. **Open Documentation** — keeping DevOps learning content transparent and accessible

---

> ⚠️ GitHub raw URLs are suitable for personal blog traffic.

> For high-traffic production use, a dedicated CDN (Cloudflare / Bunny.net) is recommended.
